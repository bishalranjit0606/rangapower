# Ranga Power

A simple, fun static website that suggests a workout split based on how many days you can hit the gym.  
Choose your available days, and the site instantly shows a fitting routine from 1-day full body workouts to multi-day splits.

Check out the live site here: [Ranga Power](https://bishalranjitkar.tech/rangapower/)

## Features
- Select how many days you can train  
- Automatically get a recommended workout split  
- Covers full-body, upper/lower, push–pull–legs, and more  
- Lightweight, fast, and fully static  
- Simple and clean UI

## Tech Stack
- HTML
- CSS
- JavaScript
- 100% static — no backend required

## How to Use
1. Open the website in your browser.  
2. Choose how many days you can go to the gym this week.  
3. Get your recommended workout routine.  
4. Start training!
